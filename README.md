
# Aviation Fuel LCA Dashboard

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## What it includes

- Overview of the aviation fuel LCA model
- Editable fuel pathway table
- Scenario comparison by distance class
- Scenario ranking
- Scenario matrix and system boundary summary

## Notes

The default numbers are preloaded from the values discussed in the report draft and the uploaded SimaPro comparison outputs. You can replace any scenario values directly in the app code or extend the data tables.
